**每次写提交脚本必须检查**：
1. 读取 npz 概率图后，**不要转置**
2. 读取 nii.gz 分割结果后，**不要转置**
3. 用 `grep -r "np.transpose(pred" active/` 检查所有脚本

**验证方法**：测试两种方式，选择 Dice 更高的
```python
dice_no_trans = dice_score(pred, label)

# 通常 no_trans 更高！
```

### 评估指标 (重要!)
```
Score = 0.30 * TopoScore + 0.35 * SurfaceDice + 0.35 * VOI_score
```
- **SurfaceDice@2**: 表面边界相似度 (容差2个体素)
- **VOI_score**: 实例分割一致性 (避免分裂/合并)
- **TopoScore**: 拓扑特征保持 (Betti数匹配)

**注意**: 不是简单的 Dice Score! 需要保持拓扑连续性
